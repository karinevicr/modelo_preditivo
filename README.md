# Desafio Kaggle - Machine Learning

Este projeto é parte de um desafio no Kaggle, onde apliquei meus conhecimentos de **Machine Learning** para resolver um problema real e aprender mais sobre os diversos algoritmos e técnicas dessa área.

## Objetivo do Projeto

O objetivo deste projeto foi desenvolver modelos de **machine learning** para solucionar um desafio de saber qual musica teria mais popularidade. Durante o desenvolvimento, explorei diferentes abordagens, ajustei parâmetros e testei diversos algoritmos para otimizar os resultados.

## Tecnologias Utilizadas
- **Python** – Linguagem principal para desenvolvimento e experimentação.
- **Scikit-learn** – Biblioteca essencial para construção e avaliação de modelos de machine learning.
- **Pandas** – Usada para manipulação e análise de dados.
- **Matplotlib e Seaborn** – Bibliotecas para visualização de dados e resultados.

## Status do Projeto
✅ **Finalizado** ✅  
O projeto foi concluído com sucesso, e o modelo foi avaliados com base em diferentes métricas de desempenho. A análise final foi realizada e os resultados já foram apresentados no Kaggle.
